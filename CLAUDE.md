# CLAUDE.md — rindo-career-lp

このリポジトリは、株式会社RINDO（林業専門求人サイト RINDO）の**林業キャリア無料相談LP**です。
Netlify上で静的サイトとして公開されています（WordPressではありません）。

## このLPの目的

無料オンライン面談（Google Meet）の**予約数を増やすこと**が唯一の目的です。
転職を決めていない段階の未経験者・経験者にも、気軽に予約・相談してもらえることを重視しています。

編集・改善の提案をするときは、常に「これは予約数を増やす方向か？」を判断基準にしてください。

## 必ず守ること

1. **スマホ表示を最優先すること。** 訪問者の大半はスマホからのアクセスです。PC表示だけを見て「良さそう」と判断しないでください。
2. **RINDOらしい落ち着いた森林系トーンを維持すること。** `style.css` の `--forest-*` `--ink-*` `--paper-*` 系のカラートークンを使い、派手な配色・急な世界観の変更をしないでください。
3. **予約導線を壊さないこと。** 以下は絶対に削除・変更しないでください。
   - `index.html` 内の Spir予約ウィジェット（`<div class="spir-widget" data-url="...">`）
   - Spirの外部スクリプト（`<script src="https://app.spirinc.com/js/external/iframe.js" async>`）
   - LINEリンク（`https://lin.ee/My8T9kC`）
   - ページ内リンク `#reserve`（予約セクションへのアンカー）
4. **APIキーやパスワードなどの機密情報をHTML/CSS/JSに直接書かないこと。** 外部サービスとの連携が必要な場合は、環境変数や外部設定で管理する方法を提案してください。
5. **不要なライブラリ・フレームワークを追加しないこと。** このLPは素のHTML/CSSのみで構成されたシンプルな静的サイトです。React・Vue・jQuery・ビルドツール（webpack等）を安易に導入しないでください。
6. **WordPress前提のコード（PHPタグ、WordPressショートコード、テーマ関数呼び出しなど）に変換しないこと。** このLPは単体HTMLとしてNetlifyで公開する運用です。
7. **画像・CSSのパスは相対パスのまま維持すること。** `assets/...`、`style.css` のようなNetlify上でそのまま機能する相対パスを使ってください。ローカル環境依存の絶対パスを書かないでください。

## 変更後に必ず提示すること

コードを変更したら、作業報告の最後に必ず以下を明示してください。

- 何を、なぜ変更したか
- 予約ウィジェット・LINEリンク・`#reserve`リンクに影響がないことの確認
- スマホ表示で崩れないかの確認ポイント
- ユーザー（非エンジニア）が公開前にブラウザで確認すべきポイント（README.mdの「公開前チェックリスト」参照）
